# 将棋 仕様

## 概要

CPU 対戦専用の将棋。人間 vs CPU の 1 対 1。
パス&プレイ (対人) は不採用。

---

## ゲームルール

標準将棋のルールに準拠:
- 9×9 盤面
- 先手・後手を選択して対局開始
- 詰み → 詰まされた側の負け
- 投了機能あり

---

## 難易度 (CPU の強さ)

| レベル | 内容 |
|--------|------|
| 弱 | 駒得のみ評価 |
| 普通 | 囲いを作る |
| 強 | 定跡 + 深読み |

エンジン: `SimpleMinimaxEngine` (ミニマックス法)
定跡: `OpeningBook.swift`

---

## 操作

- マスをタップ → 駒を選択
- 選択後に目的マスをタップ → 移動
- 持ち駒エリアの駒をタップ → 打ち駒選択
- 成り選択: 成・不成のどちらかを選ぶオーバーレイが出る

---

## UI レイアウト (縦順)

1. ステータスバー (手番 / 直前手 / 待ったボタン / 投了ボタン / 手数)
2. CPU の持ち駒
3. 盤面 (正方形)
4. あなたの持ち駒
5. 検討コントロール (終局後のみ)
6. バナー広告

人間が後手の場合、盤面を 180° 反転表示。

---

## 持ち駒表示

- 五角形の駒形 (KomaView, 32pt) を横並び表示
- 枚数が多い場合は横スクロール対応
- 枚数を駒の下に「×N」で表示
- 選択中の駒はハイライト（黄色背景）
- CPU 側の駒は上向き (自陣方向)、あなたの駒は下向き

---

## 待った機能

- 直前の「自分の手 + CPU の応手」2 手をまとめて巻き戻す
- **無料: 1 局につき 1 回**
- 2 回目以降はインタースティシャル広告を視聴してから実行
- 使用状況はスナップショットに永続化（アプリ再起動後も無料枠は消費済み）
- 確認アラート表示あり

---

## 投了機能

- ステータスバーの「投了」ボタンをタップ
- 確認アラート → 「投了する」で即終了
- 結果: 「あなたの負け（投了）」
- 投了後は検討モードに移行（KIF エクスポート可）

---

## 終局後 (検討モード)

- 手を 1 手ずつ戻す・進める
- KIF 形式でエクスポート（ShareSheet）

---

## 永続化 (ShogiSnapshot)

| フィールド | 内容 |
|-----------|------|
| initialSfen | 開始局面 |
| moves | USI 形式の指し手列 |
| phase | playing / review |
| reviewPly | 検討中の手数 |
| sente / gote | human / ai |
| aiLevel | 難易度 |
| startedAt | 対局開始日時 |
| undoUsed | 待った使用済みフラグ |
| resigned | 投了フラグ |
