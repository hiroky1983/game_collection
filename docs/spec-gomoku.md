# 五目並べ 仕様

## 概要

CPU 対戦専用の五目並べ (15×15 盤)。

---

## ゲームルール

- 15×15 盤面
- 黒 (先手) / 白 (後手) を選択
- 縦・横・斜めいずれかに 5 つ連続で並べると勝利
- 禁手なし (三三・四四・長連のルールなし)
- 引き分け: 盤面が全て埋まった場合

---

## 難易度 (CPU の強さ)

| レベル | 内容 |
|--------|------|
| 弱 | 1 手読み |
| 普通 | 2 手読み |
| 強 | 3 手読み |

エンジン: `SimpleGomokuEngine`

---

## 操作

- 盤面をタップ → 最近傍の交点に石を置く

---

## UI レイアウト (縦順)

1. ステータスバー (手番 / 勝敗表示 / 待ったボタン / 投了ボタン / 手数)
2. CPU の石情報行 (石の色 + 名前)
3. 盤面 (正方形、木目調)
4. あなたの石情報行
5. バナー広告

---

## 待った機能

将棋と同仕様:
- 直前の「自分の手 + CPU の応手」2 手をまとめて巻き戻す
- **無料: 1 局につき 1 回**
- 2 回目以降はインタースティシャル広告視聴が必要
- 確認アラート表示あり

---

## 投了機能

- ステータスバーの「投了」ボタンをタップ
- 確認アラート → 「投了する」で即終了
- 結果: CPU の勝ち (`winner = humanSide.opponent`)

---

## 永続化 (GomokuSnapshot)

| フィールド | 内容 |
|-----------|------|
| cells | 盤面セル状態 |
| currentStone | 現在の手番 |
| humanSide | 人間の石色 |
| aiLevel | 難易度 |
| startedAt | 対局開始日時 |
| moveHistory | 手順リスト |
| undoUsed | 待った使用済みフラグ |
| resigned | 投了フラグ |
| winner | 勝者 (通常勝利・投了どちらも保存) |

`winner` を保存することで、画面離脱後に戻っても勝利状態が正しく復元される。
旧スナップショット (`winner` なし) は `resigned` フラグで補完（後方互換）。
