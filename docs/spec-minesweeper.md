# マインスイーパー 仕様

## 概要

地雷を踏まずに全マスを開けるパズルゲーム。ソロプレイ。

---

## ゲームルール

- 数字 = 周囲 8 マスにある地雷の数
- 数字 0 のマスを開けると隣接する空白が連鎖開放 (BFS 方式)
- 地雷を踏むとゲームオーバー
- 地雷以外の全マスを開けるとクリア
- **初手で絶対に地雷を踏まない** (初タップの周囲 3×3 を安全ゾーンとして地雷配置)

---

## 難易度

| レベル | 盤面 | 地雷数 |
|--------|------|--------|
| 初級 | 9×9 | 10 |
| 中級 | 12×12 | 25 |
| 上級 | 15×15 | 40 |

ゲーム開始時に難易度選択シートを表示。ゲーム中に「新規対局」ボタンでも変更可能。

---

## 操作

| 操作 | 動作 |
|------|------|
| タップ | マスを掘る (フラグモード OFF 時) |
| タップ | 旗を立てる/外す (フラグモード ON 時) |
| 長押し (0.35 秒) | 旗を立てる/外す (常時有効) |

---

## UI

### ステータスバー

```
[🚩 残り地雷数]  [😊/😎/😵 リスタートボタン]  [🕐 タイマー] [🚩 フラグモードトグル]
```

- タイマー: 初手から計測、ゲームオーバー / クリアで停止
- フラグモード: ON 時はタップで旗を立てる操作になる

### 盤面

- グレー背景 (`#777777`)
- 未開放マス: `#BDBDBD`
- 開放済みマス: `#D8D8D8`
- 地雷を踏んだマス: `Theme.coral` (赤橙)
- 数字色: 1=青, 2=緑, 3=赤, 4=濃青, 5=暗赤, 6=シアン, 7=黒, 8=グレー

### コンティニューオーバーレイ

地雷を踏んだ際に盤面上に表示:

```
💥 地雷を踏んだ！
[広告を見てコンティニュー]
[あきらめる]
```

---

## コンティニュー仕様

- **回数制限なし** (踏むたびに広告を見ればコンティニュー可能)
- インタースティシャル広告視聴 → `continueAfterAd()` を呼び出し
- コンティニュー時の処理:
  1. ゲームオーバーで露出していた地雷を全て再び隠す
  2. 踏んだ地雷のマスに旗を立てる (既知の危険マスとして残す)
  3. タイマー再開、`gameState = .playing` に復帰
- 「あきらめる」→ 新規ゲームシートへ

---

## 永続化

現バージョンではスナップショット保存なし。
アプリ離脱時に途中状態は失われる。
